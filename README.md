# 3 HouseHub (3HB) — Site Institucional

Site multi-páginas da **3 HouseHub**, com identidade visual tecnológica e design inspirado em referências premium B2B.

## Páginas

| Rota | Conteúdo |
|------|----------|
| `/` | Hero tech, parceiros, diferenciais, CTA |
| `/sobre` | Missão, visão, valores, timeline, equipe |
| `/servicos` | 6 serviços com cards interativos |
| `/contato` | Formulário + informações + mapa |

## Identidade visual

| Cor | Hex | Uso |
|-----|-----|-----|
| Azul Marinho | `#0D2A56` | Fundo principal |
| Azul Médio | `#1E5BAA` | Gradientes, destaques |
| Laranja | `#FF5A1F` | CTAs, acentos |
| Branco | `#FFFFFF` | Textos |

**Fonte:** Exo 2

## Como rodar

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
